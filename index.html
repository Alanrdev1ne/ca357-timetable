Accessible Timetable

This website was built to provide an easy to use, lightweight and accessible alternative to the current DCU timetable website. The website is built with both mobile and desktop use in mind.
The site uses HTML, CSS and bootstrap to keep it stylish, yet lightweight. This also makes it much faster and more responsive than the current website which uses JavaScript frameworks.
